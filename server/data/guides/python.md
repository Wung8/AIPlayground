## Download

Go to [python.org/downloads](https://www.python.org/downloads) and download the latest Python 3 installer.

## Install (Windows)

1. Run the installer
2. **Check "Add Python to PATH"** before clicking Install Now
3. Click Install Now

## Verify

Open a terminal and run:

```
python --version
```

You should see something like `Python 3.11.2`. If you get an error, Python was not added to PATH. Re-run the installer and make sure to check the Add Python to PATH box.
